
# Shadcn Loaders

A comprehensive collection of 20+ professional React loaders and spinners inspired by shadcn/ui. Built with Tailwind CSS, supporting both Dark and Light modes.

## Installation

```bash
npx shadcn-loaders@latest add loader
```

## Features

- **20+ Variants**: Modern and performant animations.
- **Theme Support**: Fully compatible with Tailwind's dark mode.
- **Customizable**: Control sizes and colors via props.
